# Code Of Conduct

This project follows a simple standard:

- be respectful
- be direct
- assume good intent
- critique code and behavior, not people
- avoid harassment, discrimination, or hostile conduct

## Expected Behavior

- Give technical feedback clearly and specifically.
- Keep disagreements grounded in evidence.
- Respect maintainers' time and the repository scope.
- Help make the project safer and more understandable for other contributors.

## Unacceptable Behavior

- Personal attacks
- Harassment or intimidation
- Hate speech or discriminatory behavior
- Deliberate disruption, spam, or bad-faith contribution patterns

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors whose behavior violates this standard.
